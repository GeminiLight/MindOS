These assets are used only for Inbox link source attribution. They must not be used in marketing copy, partnership claims, platform support badges, or call-to-action buttons.

## Neutral Badges

The UI intentionally uses generated text badges for sources without a clear public brand-kit permission trail in this product context:

- Xiaohongshu: `小`
- Zhihu: `知`
- WeChat public articles: `微`
- arXiv: `ar`
- Reddit: `r/`

## Rules

- Do not hotlink third-party logos at runtime.
- Do not source official assets from third-party icon sites.
- Do not recolor, redraw, distort, add effects to, or crop official marks.
- Keep platform detection separate from icon assets so unclear platforms can stay on neutral badges.
- Use these icons only next to captured source links and saved Inbox rows.
