# 🍪 小红书

供 Mac 本地的 `xiaohongshu-mcp` 使用，扫码登录后生成 `cookies.json`。

## 获取 cookies.json

```bash
# Apple Silicon（M 系列）
wget https://github.com/xpzouying/xiaohongshu-mcp/releases/latest/download/xiaohongshu-mcp-darwin-arm64.tar.gz
tar -xzf xiaohongshu-mcp-darwin-arm64.tar.gz

# 运行登录，扫码完成后生成 cookies.json
./xiaohongshu-login
```

## 上传到服务器

```bash
scp cookies.json <user>@<server>:~/.local/bin/cookies.json
```

`cookies.json` 需与服务器上的 `xiaohongshu-mcp` 二进制在同一目录（`~/.local/bin/`）。
