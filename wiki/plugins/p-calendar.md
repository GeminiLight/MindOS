# Calendar — 日历视图插件

> 按日期文件名渲染月历视图，点击日期跳转到对应笔记。

## 基本信息

| 字段 | 值 |
|------|---|
| ID | `calendar` |
| 类型 | 渲染器（Renderer） |
| 触发文件 | 全局视图切换（不绑定单一文件） |
| 依赖 | 无额外依赖 |
| 状态 | 计划中 |

## 解决什么问题

日记/日志用户按 `YYYY-MM-DD.md` 命名文件，但在文件树中只能看到一长串列表。日历视图让用户直观地看到哪些天写了笔记、笔记密度如何，快速跳转到某一天。

## 功能

- 月历网格视图：有笔记的日期显示圆点标记
- 笔记密度热力图：写得越多颜色越深
- 点击日期 → 打开对应 `YYYY-MM-DD.md`
- 如果当天没有笔记 → 点击自动创建
- 支持多种日期文件名格式：
  - `YYYY-MM-DD.md`
  - `YYYY-MM-DD-标题.md`
  - `Daily/YYYY-MM-DD.md`（子目录）
- 月份切换：左右箭头或滑动
- 侧边预览：hover 日期显示笔记摘要

```
 ● = 有笔记
 ●● = 深色 = 笔记内容多
```

## 实施要点

- 扫描知识库中匹配日期模式的文件名（不需要读文件内容）
- 可配置扫描目录（默认全库，可限定为 `Daily/` 等）
- 性能：文件列表缓存，不需要实时扫描
